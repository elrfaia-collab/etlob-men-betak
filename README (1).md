## 🚀 رفع على GitHub Pages (خطوة بخطوة)

### الخطوة 1 — إنشاء حساب GitHub
1. روح على [github.com](https://github.com)
2. اضغط **Sign up** وأنشئ حساب مجاني

### الخطوة 2 — إنشاء Repository جديد
1. بعد الدخول، اضغط الزر الأخضر **New** (أو علامة +)
2. في **Repository name** اكتب: `etlob-men-betak`
3. اختار **Public**
4. اضغط **Create repository**

### الخطوة 3 — رفع الملفات
1. في الصفحة الجديدة، اضغط **uploading an existing file**
2. اسحب الملفات دي وحطها:
   - `etlob_men_betak_client.html`
   - `etlob_men_betak_admin.html`
   - `logo.png` (لو عندك الصورة)
3. في أسفل الصفحة اضغط **Commit changes**

### الخطوة 4 — تفعيل GitHub Pages
1. في نفس الـ Repository، اضغط **Settings** (فوق)
2. من القائمة الجانبية اضغط **Pages**
3. في **Source** اختار **Deploy from a branch**
4. في **Branch** اختار **main** ثم **/ (root)**
5. اضغط **Save**

### الخطوة 5 — رابط موقعك 🎉
بعد دقيقتين، هيظهر رابط زي كده:
```
https://اسم-حسابك.github.io/etlob-men-betak/
```

روح على الرابط ده:
- **العميل:** `/etlob_men_betak_client.html`
- **الأدمن:** `/etlob_men_betak_admin.html`
